# 💰 Controle Financeiro Pessoal

Um sistema simples e eficiente para gerenciamento de gastos pessoais, permitindo controle mensal, categorização de despesas e visualização de dados em gráficos.

---

## 🚀 Funcionalidades

- ✅ Cadastro de gastos
- ✅ Definição de receita mensal
- ✅ Cálculo automático de saldo
- ✅ Filtro de gastos por mês
- ✅ Categorização de despesas
- ✅ Marcar gastos como pagos ou pendentes
- ✅ Exclusão de gastos
- ✅ Gráfico de categorias (Chart.js)
- ✅ Modo escuro 🌙

---

## 🖥️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js
- Firebase (em integração 🔥)
