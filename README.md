🚀 **PATCH NOTES — Atualização do Sistema de Gestão Financeira**

💥 Nova versão liberada com várias melhorias e evoluções importantes no sistema!

---

## 🔐 Autenticação

* Sistema de login com e-mail e senha
* Cadastro de novos usuários
* Login com Google integrado
* Exibição do nome do usuário na interface
* Logout funcional
* Proteção do sistema com tela de autenticação

---

## ☁️ Banco de Dados

* Integração com Firebase Firestore
* Dados salvos na nuvem
* Informações vinculadas a cada usuário
* Sincronização automática entre dispositivos

---

## 💰 Gestão de Gastos

* Adição de gastos com:

  * Descrição
  * Valor
  * Categoria
  * Data
  * Status (pendente/pago)
* Exclusão de gastos
* Marcar gastos como pagos

---

## 📅 Controle Mensal

* Filtro de gastos por mês
* Receita mensal separada
* Cálculo automático baseado no mês selecionado

---

## 📊 Resumo Financeiro

* Exibição de:

  * Receita total
  * Total gasto
  * Saldo final
* Indicador visual (verde/vermelho) conforme saldo

---

## 📈 Gráficos

* Gráfico de barras (Receita x Gasto)
* Gráfico de pizza (Gastos por categoria)
* Atualização automática em tempo real

---

## 🧾 Interface de Gastos

* Layout em cards modernos
* Status visual para cada gasto
* Botões de ação (Excluir / Pagar)
* Confirmação antes de excluir

---

## 🎨 Interface (UI/UX)

* Layout organizado e responsivo
* Animações e efeitos visuais
* Melhor distribuição de elementos
* Design mais profissional

---

## 🌙 Modo Escuro

* Alternância entre tema claro/escuro
* Preferência salva automaticamente
* Interface e gráficos adaptados

---

## 🧠 Categorias (GRANDE ATUALIZAÇÃO 🚀)

* Sistema de categorias dinâmicas
* Usuário pode criar novas categorias
* Salvamento automático no banco
* Carregamento ao iniciar o sistema
* Prevenção de categorias duplicadas

---

## 🔄 Sincronização

* Todas as ações são salvas automaticamente
* Dados carregados ao fazer login
* Atualização instantânea da interface
