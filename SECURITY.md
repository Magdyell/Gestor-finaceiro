# Security Policy

## Supported Versions

Atualmente, apenas a versão mais recente deste projeto está recebendo atualizações de segurança.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

Se você encontrar uma vulnerabilidade de segurança neste projeto, por favor, não abra uma issue pública.

Em vez disso, entre em contato de forma privada pelo e-mail:

uynsksh@gmail.com

Ao relatar uma vulnerabilidade, tente incluir:
- uma descrição clara do problema
- os passos para reproduzir
- o possível impacto
- prints ou trechos de código, se necessário

Farei o possível para responder em até **7 dias** com uma atualização inicial.

Se a vulnerabilidade for confirmada, ela será analisada e corrigida assim que possível.
Se não for considerada uma vulnerabilidade de segurança, eu também retornarei com uma explicação.
